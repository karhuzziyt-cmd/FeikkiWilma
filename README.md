# FeikkiWilma
Feikki wilman koodi, vapaasti muokattavissa
FeikkiWilma on open-source. 

Tee feikkejä aidon näköisiä feikkiwilma merkintöjä

tämän paketti sisältää kaikki kirjautumis logiikat ja valikot muista korvata kopioidessa minun firebase APIt omiksi api ksi koska minun API ei tule toimimaan sinun versiossasi

Älä ole skid ja kopioi koko koodia vaan voit muokata ja tehdä siitä parempaa
